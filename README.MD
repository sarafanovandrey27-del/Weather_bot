# Telegram bot (Погода сейчас)
**Описание: Данный телеграм бот был создан для полчения погодных данных в определённой точке, отправленной пользователем**

## Используемые технологии:
 1. **pyTelegramBotAPI**
2. **Requests**
3. **OpenWeatherMap API**
## Как запустить проект
1. **Клонируйте репозиторий: https://github.com/sarafanovandrey27-del/Weather_bot.git**
2. **Установите все требуемые зависимости: pip install pyTelegramBotAPI requests**
3. **Получите API ключи (в BotFather в телеграм и создайте своего нового бота и получите TOKEN, а так же в OpenWeatherMap зарегистрируйтесь и получите API-ключ)**
4. **В основном файле weather_b.py вставьте ваш токен с BotFather и API ключ с OpenWeatherMap**
5. **Запустите бота через терминал PyCharm (python weather_b.py)**
